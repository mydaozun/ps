# FreePS

在线图片编辑器允许你使用 HTML5 技术创建和编辑图片。不需要购买、下载、安装或使用过时的闪存。没有广告。主要功能：图层、滤镜、HTML5、开源、Photoshop 替代方案。

FreePS 直接在浏览器中运行。你可以创建图片，从剪贴板粘贴（ctrl+v），或者从电脑上传（使用菜单或拖放）。不会把任何东西发送到任何服务器。所有内容都留在浏览器里。
